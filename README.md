## vue-lib-hello

一个简单的vue库脚手架，fork 或 clone 本仓库，即可搭建完成一个vue库的基础框架

## 安装
```
npm install
```

### 运行
```
npm run serve
```

### 编译examples项目
```
npm run build
```

### 语法检查
```
npm run lint
```

### 编译库
```
npm run lib
```

### 发布

```bash
npm run release
git push --follow-tags origin master && npm publish
```
